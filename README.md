# ShilpaKala Android App

ShilpaKala is an Android application developed for artisans and small businesses to support digital branding and product presentation.

## Features
- Product showcase
- Image editing and branding
- Splash screen
- Modern Android UI
- Save and share branded images

## Technologies Used
- Java
- Android Studio
- XML
- Android SDK
